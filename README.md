## 教程
fork 仓库，添加 actions 。 setting 里添加 Actions secrets
- TIANQI_APPID //天气 apiid
- TIANQI_APPSEC // 天气 appsecet
- FROM_ADDR // 发件人

- FROM_PSWD // 发件人密码

- TO_ADDR // 发给谁
